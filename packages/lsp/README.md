# @vellum/lsp

CLI-native Language Server Protocol integration for Vellum.

This package provides:
- LspHub lifecycle management
- CLI-compatible LanguageClient
- Default server configs (TypeScript, Python, Go, Rust)
- Tool integrations for diagnostics, hover, definitions, references, symbols, and call hierarchy

See `docs/pre/phases/finished_docs/30-lsp-integration.md` for the full spec.
